# Vulcanix-AI
====================================================================
VULCANIX AI — Terminal Client for OpenRouter
A powerful, beautiful and flexible AI terminal client.
====================================================================

✨ FEATURES
• Beautiful orange terminal interface with large ASCII art
• Support for ANY model from OpenRouter (free and paid)
• Real‑time loading animation while thinking
• Excellent at generating clean, well‑formatted code
• Conversation history and context memory
• Easy model selection at startup
• Clean and user‑friendly English interface

====================================================================
📁 PROJECT FILES
main.py        – Main application
config.py      – API key configuration
README.md      – This documentation

====================================================================
🚀 INSTALLATION
1) Create project folder:
   mkdir vulcanix-ai && cd vulcanix-ai

2) (No external dependencies required)

====================================================================
⚙️ CONFIGURATION
Create file: config.py

   API_KEY = "sk-or-..."     # ← Replace with your OpenRouter API key

Get your API key:
   https://openrouter.ai/keys

====================================================================
💡 HOW TO USE
Run the program:
   python main.py

Select:
   1. Start

Enter the model (or press Enter for openrouter/free)

Type your queries.
Type "exit" to quit the program.
